# Change Log

All notable changes to the "dopamine-dev" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.7]

- **Cross-Window Sync:** Added a file watcher to synchronize your coin wallet balance across multiple open VS Code windows in real-time.
- **Interactive Notifications:** Added a "Spin Again (1 Coin)" button to Quote rewards, allowing you to reroll for inspiration instantly.
- **Improved Feedback:** The status bar now provides more accurate feedback for "Minor changes" vs. "Significant deletions" vs. "Zero rewards" due to bugs.
- **Monthly Analytics:** The analytics dashboard now supports month-by-month navigation, so you can view your earning history over time.
- **Quote Categories:** Added new settings (`dopamineDev.quoteCategories.*`) to enable/disable specific types of quotes (Humor, Programming, Motivation, Productivity).
- **Compatibility:** Updated engine requirements to support VS Code versions back to 1.86.0.
- **Security:** Resolved high-severity vulnerabilities in dependencies.
- **Bug Fixes:** Fixed issues with notification stacking and button interactions.
